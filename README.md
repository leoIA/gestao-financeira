# iLAB4 Gestão Financeira

Sistema de gestão financeira profissional desenvolvido em Laravel, projetado para oferecer controle rigoroso de fluxo de caixa e organização administrativa para empresas que buscam uma base sólida para crescimento.

## 📌 Visão Geral
Este sistema foca na gestão centralizada de uma única organização, permitindo o controle total de movimentações financeiras, contas a pagar/receber e relacionamento com fornecedores e clientes.

## 🔧 Funcionalidades
* **Dashboard Administrativo:** Controle visual de saldos bancários e fluxo de caixa.
* **Gestão de Lançamentos:** Fluxo simplificado de entradas e saídas de caixa.
* **Relatórios:** Relatórios de Saldo de Contas, Movimentação Financeira, Fluxo de Caixa Projetado e Análise por Clientes.
* **Cadastros Estruturados:** Gerenciamento de Clientes, Fornecedores, Bancos, Planos de Contas e Centros de Custo.
* **Controle de Contas:** Organização de contas bancárias e métodos de pagamento.
* **Segurança:** Sistema de autenticação e controle de acesso integrado.

## 💎 Módulos e Integrações (Sob Demanda)
Para manter o sistema enxuto e escalável, recursos avançados e integrações com terceiros são desenvolvidos e licenciados de acordo com a necessidade específica de cada negócio:

* **Integração NFSe (Reforma Tributária):** Desenvolvimento de módulos para emissão de nota fiscal de serviço diretamente com prefeituras ou ambiente nacional.
* **Automação de Gateways:** Integração via Webhook e API com plataformas como Asaas, Pagar.me, Hotmart e Kiwify para baixas automáticas.
* **Relatórios Estratégicos (DRE):** Implementação de Demonstrativo de Resultados para análise de lucro real.
* **Programação Financeira:** Módulo de recorrência e agendamento de contas futuras.

## 🚀 Instalação
    
Desenvolvido utilizando Laravel 7.0

Configure no seu arquivo .env as conexões com o mySQL.

No seu cmd ou terminal, na pasta do projeto, inicie a instalação do sistema.

```
composer install
```

```
php artisan key:generate
```

```
php artisan migrate
```
